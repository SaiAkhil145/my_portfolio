## ⚙️ Getting Started

<details>
<summary><b>🔧 Installation Steps (click to expand)</b></summary>

<br/>

**1. Clone the repository**
```bash
git clone https://github.com/SaiAkhil145/my_portfolio
cd my_portfolio
```

**2. Install dependencies**
```bash
npm install
```

**3. Run the development server**
```bash
npm run dev
```

**4. Open in browser**
```
http://localhost:3000
```

</details>

<details>
<summary><b>🚀 Build for Production (click to expand)</b></summary>

<br/>

```bash
npm run build
npm start
```

</details>

## 🎨 Customization Guide

<details>
<summary><b>✏️ How to Personalize (click to expand)</b></summary>

<br/>

**Update Personal Info**
```bash
components/Header.jsx    # Name, title, hero text
components/About.jsx     # Bio, skills, background
components/Contact.jsx   # Email, social links
```

**Replace Images**
```bash
/assets/images/          # Profile & project images
/public/                 # Static assets
```

**Update Resume**
```bash
public/Full_Stack_Developer__Sai_Revu_Resume.pdf   # Replace with yours
```

</details>
